The <address> tag

Display Block
Self-closing No

In order to specify someone's address, you use the <address> tag. Inside the <body> tag, the address tag marks contact information about the document itself. If it is within an <article> tag, then it stands for contact information for this specific article.

<address>
   John Dohster <br>
   Office address: <br>
   1234 Example Road <br>
   Country <br>
   Visit my website: <br>
   ex-ample.com
</address>

The text inside the address tag is usually displayed in italics. In your browser, this code will be rendered like so:


John Dohster
Office address:
1234 Example Road
Country
Visit my website:
ex-ample.com

It is best not to wrap a postal address inside an address tag unless it it part of the contact information.
