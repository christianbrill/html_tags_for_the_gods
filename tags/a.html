The <a> tag

The anchor tag is used to create links on your website. Links make up the building blocks of the internet as one page leads to another using links.

This is how you create a link:

<a href="">I am a link</a>

On your website, it will look like this:

I am a link

Usually, when you click on links, they will be opened in the same window. This happens by default. However, if you'd like to change it so when you click on a link, it is opened in a new tab, then you will have to add target="_blank" to your <a> tag, like so:

<a href="" target="_blank">I am a link</a>

If you click on the below link now, you will see that it will be opened in a new tab. Go ahead, try it.

I am a link

Anchor tags come with an annoying set of ugly default values (such as the color), so if you don't like them, I suggest you get rid of them. Especially the underline is one of the ugliest features and I like to just do away with it.

I am a link

You can turn pretty much anything into a link: words, images, buttons, areas of your website.

I am linked text.

Another very useful feature of anchor tags is the ability to send emails straight from a link. If a user clicks a link that is set up to send an email, then their default email client will be opened and they can send an email straight from there.

<a href="mailto:example@email.com">Send us an email.</a>

In your browser, it will looks just like an normal link. However, clicking on it won't open a new tab, but a new window with the email client. There's no need to set the target="_blank" attribute. Click on the below link to see what happens.

Send us an email.

Using this method, you can set not only the email address line, but also the subject line, the CC and BCC, as well as the actual email body text. In order to do that, you will have to add the following to your href attribute. Add %20 when you encounter space between words.

<a href="mailto:example@email.com?subject=Information%20Request?cc=someones@email.com,someoneelses@email.com?bcc=thirdpersons@email.com?body=Please%20send%20me%20some%20information.%20Thanks.">Send us an email that already has everything filled out for you.</a>

On your website, this will happen when you click the link:

Send us an email that already has everything filled out for you.

Yet another neat feature of the anchor tag is the ability to add a download option when a link is clicked. For example, if you have a résumé you want people to download, then you can add the href to that file and it will be downloaded as soon as it is clicked. All you have to do is add the download attribute in the opening tag of your anchor. This works with a file path to a file existing within your project as well as a URL pointing to a file existing online.

<a href="/docs/resume.pdf" download>Download my résumé here.</a>
