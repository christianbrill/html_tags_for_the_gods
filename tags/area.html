The <area> tag

The <area> tag allows you to create an area (or areas) on an image, which can either be clicked (when made into a hyperlink) or which contain some sort of other information about the image. It is only used inside the <map> tag and is self-closing.

In order to use it, you will first have an image on your page. This image is then linked to a <map> tag by using an ID. Within this map, you can then define your (clickable) areas. If you are creating a link, then you can also specify the target attribute.

<img usemap="#links" class="area_img" src="https://www.famousbirthdays.com/faces/chi-kim-image.jpg" alt="Kimchi, drag queen">
<map name="links" id="links">
    <area alt="Kimchi's Left Eye" title="Kimchi's Left Eye" href="http://google.com/search?q=kimchi+left+eye" shape="rect" coords="85,160,126,182" />
    <area alt="Kimchi's Right Eye" title="Kimchi's Right Eye" href="http://google.com/search?q=kimchi+right+eye" shape="rect" coords="156,168,201,179" />
    <area alt="Kimchi's Lips" title="Kimchi's Lips" href="http://google.com/search?q=kimchi+lips" shape="rect" coords="115,230,163,258" />
</map>
